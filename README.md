# 子远教育官方网站

## 项目概述
子远教育是一家专业的留学咨询服务机构，致力于为学生提供全方位的海外留学规划服务。本项目是子远教育的官方网站，旨在展示公司的服务项目、品牌理念和联系方式。

## 技术栈
- Next.js 14+
- React
- TypeScript
- Tailwind CSS
- Node.js

## 页面结构
- **首页**: 展示品牌形象、核心服务和价值主张
- **关于我们**: 公司背景、团队介绍
- **服务项目**: 详细的留学服务介绍
- **成功案例**: 往期优秀案例展示
- **联系我们**: 联系方式和咨询入口

## 设计特色
- 现代化响应式设计
- 优雅的专业教育风格
- 用户友好的导航体验
- 清晰的服务展示布局

## 快速开始

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```

### 构建生产版本
```bash
npm run build
```

### 运行生产服务器
```bash
npm start
```

## 目录结构
```
src/
├── components/       # 可复用组件
├── pages/           # 页面路由
│   ├── index.tsx    # 首页
│   └── _app.tsx     # 应用根组件
├── styles/          # 样式文件
└── public/          # 静态资源
```

## 功能模块
1. **导航栏**: 固定顶部导航，包含品牌标识和主要页面链接
2. **英雄区域**: 大屏展示公司核心价值和服务优势
3. **服务网格**: 展示六大核心服务项目
4. **响应式设计**: 适配各种设备屏幕尺寸

## 部署
本项目可部署在 Vercel、Netlify 或其他支持 Next.js 的平台。

## 维护
项目采用模块化开发，便于后续功能扩展和维护。
